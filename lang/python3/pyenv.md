- install pyenv
`brew install pyenv`

- confirm available versions
`pyenv install -l`

- install python3
`pyenv install 3.6.5`

- install anaconda
`pyenv install anaconda3-5.1.0`

- confirm installed versions
`pyenv versions`

- confirm local version
`pyenv local`

- unset local version
`pyenv local --unset`

- set local version
`pyenv local 3.6.5`

- confirm global version
`pyenv global `

- change global version
`pyenv global 3.6.5`
